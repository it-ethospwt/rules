cara push tag remote
```
// kirim tag ke remote
$ git push origin namatag

// kirim semua tag ke remote
$ git push origin --tags
```

cara pull tag remote
```
// ambil tag
$ git fetch origin namatag

// ambil semua tag
$ git fetch origin
```

hapus tag remote
```
$ git push --delete origin v1.0.0
```

menampilkan semua tag
```
$ git tag
```
